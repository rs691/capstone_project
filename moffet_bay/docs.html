Setup and Deployment Guide

Overview

The Moffat Bay Resort application is a web-based platform built with Django, designed to manage reservations, user accounts, and provide information about the resort.

Technologies Used

  • Django 5.1.2
  • MySQL
  • HTMX
  • Bootstrap 5
  • Python 3.9+

Prerequisites

  • Python 3.9 or higher
  • MySQL Server
  • Git

Quick Start Guide

  1. Clone the repository: 
    git clone https://github.com/yourusername/moffat-bay.git
cd moffat-bay
  2. Create and activate a virtual environment: 
    # Windows
python -m venv venv
.\venv\Scripts\activate

# Unix/Linux
python3 -m venv venv
source venv/bin/activate
  3. Install required packages: 
    pip install -r requirements.txt
  4. Configure your database:
    • Create a MySQL database.
    • Update the .env file with your database credentials.
  5. Run migrations: 
    python manage.py makemigrations
python manage.py migrate
  6. Create a superuser: 
    python manage.py createsuperuser
  7. Start the development server: 
    python manage.py runserver

    Visit http://localhost:8000 in your browser.

Registration

Create an account to access exclusive deals and make reservations:

  1. Go to the Sign Up page.
  2. Fill in your personal and contact details.
  3. Submit the form to create your account.

Login

Log in to manage your reservations:

  1. Visit the Sign In page.
  2. Enter your email and password.
  3. Click the login button to proceed.

Account Management

Manage your account settings and preferences:

  • Update personal details in Profile Settings.
  • Reset your password via the Password Reset page.
  • View past reservations in your Dashboard.

Making a Reservation

  1. Log in to your account.
  2. Go to the Reserve a Room page.
  3. Select your desired room type and dates.
  4. Confirm the details and submit your reservation.

Searching for a Reservation

After logging in, you can search for your reservations by:

  • Reservation Number
  • Last Name
  • Email Address

Payment Process

Securely complete your reservation payment:

  1. Enter your billing details.
  2. Choose a payment method (Credit/Debit Card, PayPal, etc.).
  3. Confirm the transaction and receive a booking confirmation.

Frequently Asked Questions

How do I modify or cancel my reservation?

Log in to your account, go to the Dashboard, and select the reservation you wish to modify or cancel.

What payment methods are accepted?

We accept all major credit cards, PayPal, and resort gift vouchers.

What is the cancellation policy?

Cancellations made within 48 hours of booking are fully refundable. After that, cancellation fees may apply.
